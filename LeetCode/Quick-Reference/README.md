# DSA Quick Reference Pack

This folder is built for fast interview revision.

## Files
- `blind-75-quick-qa.md`: Blind 75 problems with quick answer + explanation + complexity.
- `top-100-quick-qa.md`: Top 100 interview-style problems with short solutions.
- `dsa-pattern-cheat-sheet.md`: All major DSA patterns with mini examples.
- `dsa-algo-template-bank-python.md`: Reusable Python templates for core algorithm patterns.
- `blind-75-algo-syntax.md`: Blind 75 problem -> template id mapping.
- `top-100-algo-syntax.md`: Top 100 problem -> template id mapping.

## How to use daily (45 mins)
1. 15 mins: solve 5 from `blind-75-quick-qa.md`
2. 10 mins: map them in `blind-75-algo-syntax.md`
3. 10 mins: write code from `dsa-algo-template-bank-python.md`
4. 10 mins: revise one section from `dsa-pattern-cheat-sheet.md`

## Quick format used
- **Q**: Problem name
- **A**: Core approach
- **Why**: Short explanation
- **TC/SC**: Time and space complexity

Tip: Solve first without seeing the answer, then compare in 30 seconds.
