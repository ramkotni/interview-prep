# Deep Learning

## Overview
Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with multiple hidden layers to learn complex patterns from large datasets.

## Neural Network Components

Input Layer
↓
Hidden Layers
↓
Output Layer

Each neuron performs:
- Weighted Sum
- Activation Function
- Output

## Popular Activation Functions
- ReLU
- Sigmoid
- Tanh
- Softmax

## Types of Neural Networks

### ANN
General-purpose neural network.

### CNN
Used for image processing.
Applications:
- Face Recognition
- Medical Imaging

### RNN
Processes sequential data.
Applications:
- NLP
- Time Series

### LSTM
Improved RNN for long-term dependencies.

### Transformers
Current state-of-the-art for NLP and Generative AI.

Examples:
- GPT
- BERT
- LLaMA

## Training Process

1. Forward Propagation
2. Loss Calculation
3. Backpropagation
4. Weight Update (Gradient Descent)
