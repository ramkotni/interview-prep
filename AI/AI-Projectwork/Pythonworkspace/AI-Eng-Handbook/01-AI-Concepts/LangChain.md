# LangChain

## Overview
LangChain is a framework for building applications powered by Large Language Models (LLMs). It simplifies prompt management, retrieval, agents, memory, and tool integration.

## Core Components

- LLMs
- Prompts
- Chains
- Retrievers
- Memory
- Tools
- Agents
- Output Parsers
