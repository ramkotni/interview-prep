# Machine Learning (ML)

## Overview
Machine Learning is a subset of Artificial Intelligence (AI) that enables computers to learn patterns from data and make predictions or decisions without being explicitly programmed.

## Types of Machine Learning

### 1. Supervised Learning
- Uses labeled data.
- Learns mapping between input and output.
- Examples:
  - Classification
  - Regression

Algorithms:
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- SVM
- XGBoost

### 2. Unsupervised Learning
- Uses unlabeled data.
- Discovers hidden patterns.

Algorithms:
- K-Means
- DBSCAN
- PCA
- Hierarchical Clustering

### 3. Reinforcement Learning
- Learns through rewards and penalties.
- Used in robotics, gaming, and autonomous systems.

## ML Workflow

1. Collect Data
2. Clean Data
3. Feature Engineering
4. Split Train/Test
5. Train Model
6. Evaluate Model
7. Hyperparameter Tuning
8. Deploy Model
9. Monitor Performance

## Common Metrics

Classification:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

Regression:
- MAE
- MSE
- RMSE
- R² Score

## Common Libraries
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- XGBoost
- LightGBM

## Applications
- Fraud Detection
- Recommendation Systems
- Spam Detection
- Predictive Analytics
- Customer Segmentation

## Key Takeaways
- Data quality is more important than model complexity.
- Feature engineering greatly impacts performance.
- Model evaluation prevents overfitting.
