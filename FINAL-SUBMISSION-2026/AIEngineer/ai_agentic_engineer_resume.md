## Part 3: Top 20 AI Agentic Engineer Interview Questions

1.  **LangGraph State Management**: *How do you manage state transitions, loops, and conditional routing inside a LangGraph application? How do you prevent infinite loops in agent planning?*
2.  **Human-in-the-Loop (HITL)**: *Explain how you implement interrupt-driven human-in-the-loop validation inside a LangGraph state machine (e.g., for invoice approvals).*
3.  **Vector Search Performance**: *How do you optimize retrieve performance in pgvector or a dedicated vector database? Explain the trade-offs between HNSW and IVFFlat index structures.*
4.  **Prompt Hacking Defense**: *How do you protect your agent's system prompts from prompt injection and jailbreak attacks, especially when executing user-provided queries?*
5.  **Agent Tool-Calling Latency**: *When an agent has access to 15+ external tools, LLM latency increases. How do you design tool selection routing or leverage Model Context Protocol (MCP) to minimize latency?*
6.  **RAG Chunking Strategies**: *Describe the differences between character-based, semantic-based, and document-structure chunking for PDF medical records. When is document parsing superior?*
7.  **Deterministic Agent Guardrails**: *Agents can hallucinate actions. How do you enforce structured JSON output formats using tools like Pydantic, Instructor, or Outlines?*
8.  **LLM Cost & Token Optimization**: *What strategies do you employ in production to reduce token usage and costs? (e.g., prompt caching, context compression, model routing).*
9.  **Evaluation of RAG (Retrieval vs. Generation)**: *How do you evaluate retrieval quality (Faithfulness, Answer Relevance, Context Recall) in CI/CD before shipping agents to staging?*
10. **Memory Architectures**: *How do you structure long-term memory (e.g., episodic, semantic) versus short-term conversation history for user profiles in multi-session agents?*
11. **FastAPI Concurrency**: *How do you structure FastAPI endpoints to handle streaming responses (Server-Sent Events) from asynchronous agents running concurrently?*
12. **Self-Correction Loops**: *How do you implement a self-correction architecture where a coding agent receives test execution exceptions and rewrites code autonomously?*
13. **Local LLMs vs. Cloud APIs**: *When would you choose Ollama (hosting Llama 3 locally) over commercial APIs like OpenAI GPT-4o? What are the infrastructure cost implications?*
14. **Observability & Tracing**: *How do you set up observability tracing (e.g., using LangSmith or OpenTelemetry) to pinpoint where an agent loop is bottlenecked or failing?*
15. **Distributed SRE for AI**: *How do you handle rate-limiting and connection pooling when an agent loops call external APIs and databases concurrently under load?*
16. **Event-Driven AI Workflows**: *How do you coordinate LangGraph agent execution asynchronously using event-driven tools like GCP Pub/Sub or RabbitMQ?*
17. **Model Context Protocol (MCP)**: *Explain the architecture of Model Context Protocol (MCP) and how it standardizes agent access to file systems, databases, and APIs.*
18. **Prompt Versioning & Registry**: *How do you manage and register changes to system prompts across dev, staging, and production without redeploying the core application?*
19. **Fine-Tuning vs. RAG**: *In what scenario is fine-tuning a small open-source model (e.g., Mistral-7B) superior to prompt-based RAG on a larger model?*
20. **Security & Data Privacy**: *How do you ensure PII data is masked or anonymized before sending context windows to external LLM providers?*
